## 一、点状分布测度

### 1.1 平均中心（Mean Center）

```python
def mean_center(self):
    """计算平均中心坐标"""
    if self.n == 0:
        return (0, 0)
    mean_x = np.mean(self.points[:, 0])
    mean_y = np.mean(self.points[:, 1])
    return (mean_x, mean_y)
```

**原理**：所有点坐标的算术平均值

$$\bar{X} = \frac{\sum_{i=1}^{n} X_i}{n}, \quad \bar{Y} = \frac{\sum_{i=1}^{n} Y_i}{n}$$

### 1.2 中项中心（Median Center）

```python
def median_center(self):
    """计算中项中心坐标"""
    if self.n == 0:
        return (0, 0)
    median_x = np.median(self.points[:, 0])
    median_y = np.median(self.points[:, 1])
    return (median_x, median_y)
```

**原理**：所有点坐标的中位数，对极端值不敏感

### 1.3 人口重心（Weighted Mean Center）

```python
def weighted_mean_center(self, weights):
    """计算加权平均中心（人口重心）"""
    if self.n == 0 or len(weights) != self.n:
        return (0, 0)
    total_weight = np.sum(weights)
    if total_weight == 0:
        return (0, 0)
    weighted_x = np.sum(self.points[:, 0] * weights) / total_weight
    weighted_y = np.sum(self.points[:, 1] * weights) / total_weight
    return (weighted_x, weighted_y)
```

**原理**：以人口或其他属性为权重的加权平均

$$X_p = \frac{\sum_{i=1}^{n} P_i \cdot X_i}{\sum_{i=1}^{n} P_i}, \quad Y_p = \frac{\sum_{i=1}^{n} P_i \cdot Y_i}{\sum_{i=1}^{n} P_i}$$

### 1.4 最近邻指数（NNI）

```python
def nearest_neighbor_index(self, area):
    """计算最近邻指数"""
    if self.n < 2:
        return (0, 0, 0)
    
    # 使用KDTree查找最近邻
    tree = KDTree(self.points)
    distances, _ = tree.query(self.points, k=2)  # k=2获取最近的两个点
    nearest_distances = distances[:, 1]  # 排除自身
    
    observed_mean_distance = np.mean(nearest_distances)
    density = self.n / area
    expected_mean_distance = 1 / (2 * np.sqrt(density))
    
    nni = observed_mean_distance / expected_mean_distance
    return (nni, observed_mean_distance, expected_mean_distance)
```

**原理**：

| NNI值 | 分布模式 |
|-------|----------|
| NNI > 1 | 均匀分布 |
| NNI = 1 | 随机分布 |
| NNI < 1 | 聚集分布 |

## 二、线状分布测度

### 2.1 网络维度（分形维数）

```python
def network_dimension(self):
    """计算网络维度（近似分形维数）"""
    if len(self.lines) == 0:
        return 0.0
    
    # 简单近似：当道路网络密集时，分形维数接近2
    total_length = sum(line.length for line in self.lines)
    # 实际应用中需要更复杂的盒计数法
    return 2.0  # 简化近似
```

**原理**：分形维数D表示空间填充能力：
- D ≈ 1：线性结构
- D ≈ 2：平面填充结构

### 2.2 迂回指数（Detour Index）

```python
def detour_index(self):
    """计算迂回指数"""
    detour_sum = 0
    count = 0
    
    for line in self.lines:
        coords = list(line.coords)
        if len(coords) >= 2:
            start = Point(coords[0])
            end = Point(coords[-1])
            straight_dist = start.distance(end)
            if straight_dist > 0:
                detour_sum += line.length / straight_dist
                count += 1
    
    return detour_sum / count if count > 0 else 1.0
```

**原理**：实际路径长度与直线距离的比值

$$DI = \frac{\sum_{i=1}^{n} \frac{L_i}{D_i}}{n}$$

- DI ≈ 1：道路较直
- DI > 1：道路曲折

## 三、面状分布测度

### 3.1 延伸率（Elongation Ratio）

```python
def elongation_ratio(self):
    """计算延伸率"""
    if self.area <= 0:
        return 0.0
    
    all_coords = []
    if isinstance(self.polygon, MultiPolygon):
        for poly in self.polygon.geoms:
            all_coords.extend(list(poly.exterior.coords))
    else:
        all_coords = list(self.polygon.exterior.coords)
    
    points = np.array(all_coords)
    pca = PCA(n_components=2)
    pca.fit(points)
    eigenvalues = pca.explained_variance_
    
    if eigenvalues[0] > 0:
        return np.sqrt(eigenvalues[1] / eigenvalues[0])
    return 0.0
```

**原理**：通过PCA提取主成分，短轴与长轴之比

$$ER = \frac{\lambda_{min}}{\lambda_{max}}$$

### 3.2 紧凑指数（Compactness Index）

```python
def compactness_index(self):
    """计算紧凑指数"""
    if self.perimeter <= 0:
        return 0.0
    ideal_perimeter = 2 * np.sqrt(np.pi * self.area)
    return ideal_perimeter / self.perimeter
```

**原理**：等面积圆周长与实际周长之比

$$CI = \frac{2\sqrt{\pi A}}{P}$$

- CI ≈ 1：接近圆形（紧凑）
- CI < 1：形状不规则

### 3.3 圆形率（Circularity Ratio）

```python
def circularity_ratio(self):
    """计算圆形率"""
    if self.perimeter <= 0:
        return 0.0
    return (4 * np.pi * self.area) / (self.perimeter ** 2)
```

**原理**：基于面积-周长比的形状指标

$$CR = \frac{4\pi A}{P^2}$$

### 3.4 Boyce-Clark形状指数

```python
def boyce_clark_shape_index(self):
    """计算Boyce-Clark形状指数"""
    if self.area <= 0:
        return 0.0
    
    centroid = self.polygon.centroid
    cx, cy = centroid.x, centroid.y
    
    # 生成36条等角度射线
    num_rays = 36
    angles = np.linspace(0, 2 * np.pi, num_rays, endpoint=False)
    
    distances = []
    for angle in angles:
        ray_length = 10.0
        end_x = cx + ray_length * np.cos(angle)
        end_y = cy + ray_length * np.sin(angle)
        ray = LineString([(cx, cy), (end_x, end_y)])
        
        intersection = self.polygon.intersection(ray)
        # 处理交点（Point/LineString/MultiPoint/MultiLineString）
        # ... 获取最远交点距离
        
    distances = np.array(distances)
    mean_dist = np.mean(distances)
    
    if mean_dist > 0:
        bc_index = np.sum(np.abs(distances - mean_dist) / mean_dist) / len(distances)
    return bc_index
```

**原理**：从质心向边界作等角度射线，测量距离变异程度

$$S = \frac{1}{n} \sum_{i=1}^{n} \left| \frac{d_i - \bar{d}}{\bar{d}} \right|$$

- S ≈ 0：接近圆形
- S 越大：形状越不规则

## 四、核心技术要点

### 4.1 几何运算库 - Shapely

| 类/方法 | 用途 |
|---------|------|
| `Point(x, y)` | 创建点对象 |
| `LineString(coords)` | 创建线对象 |
| `Polygon(coords)` | 创建多边形 |
| `MultiPolygon(polygons)` | 创建复合多边形 |
| `polygon.centroid` | 计算质心 |
| `polygon.intersection(ray)` | 几何求交 |
| `point.distance(other)` | 距离计算 |

### 4.2 空间索引 - KDTree

```python
from scipy.spatial import KDTree

# 构建空间索引
tree = KDTree(points)
# 查询最近邻（k=2包含自身）
distances, indices = tree.query(points, k=2)
```

### 4.3 主成分分析 - PCA

```python
from sklearn.decomposition import PCA

pca = PCA(n_components=2)
pca.fit(points)
eigenvalues = pca.explained_variance_  # 特征值 = 轴长度平方
```

## 六、注意事项

1. **坐标系统**：确保所有数据使用同一坐标系（如WGS84）
2. **数据质量**：检查几何对象是否有效（`polygon.is_valid`）
3. **数值精度**：处理浮点计算时注意数值稳定性
4. **边界情况**：空多边形、单一点、直线等特殊情况
5. **性能优化**：大规模数据使用空间索引加速查询
