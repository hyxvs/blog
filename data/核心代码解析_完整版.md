## 一、点状分布测度分析器

### 1.1 类定义与初始化

```python
class PointDistributionAnalyzer:
    """点状分布测度分析器"""
    
    def __init__(self, points, weights=None):
        """
        初始化
        :param points: 点坐标数组，shape (n, 2)，列顺序为 [经度, 纬度]
        :param weights: 权重数组，用于加权平均中心（如人口权重）
        """
        self.points = np.array(points)
        self.weights = np.array(weights) if weights is not None else None
        self.n = len(points)
```

### 1.2 平均中心与加权平均中心

```python
def mean_center(self):
    """计算平均中心（Mean Center）"""
    if self.weights is not None:
        # 加权平均中心（人口/经济重心）
        total_weight = np.sum(self.weights)
        mx = np.sum(self.points[:, 0] * self.weights) / total_weight
        my = np.sum(self.points[:, 1] * self.weights) / total_weight
        return (mx, my), "加权平均中心（人口/经济重心）"
    else:
        # 简单平均中心
        mx = np.mean(self.points[:, 0])
        my = np.mean(self.points[:, 1])
        return (mx, my), "平均中心"
```

**数学原理**：

$$\text{简单平均中心：} \bar{X} = \frac{\sum X_i}{n}, \quad \bar{Y} = \frac{\sum Y_i}{n}$$

$$\text{加权平均中心：} X_p = \frac{\sum P_i \cdot X_i}{\sum P_i}, \quad Y_p = \frac{\sum P_i \cdot Y_i}{\sum P_i}$$

### 1.3 中项中心

```python
def median_center(self):
    """计算中项中心（Median Center）"""
    # 中项中心是使到所有点的曼哈顿距离之和最小的点
    mx = np.median(self.points[:, 0])
    my = np.median(self.points[:, 1])
    return (mx, my), "中项中心"
```

### 1.4 最近邻指数（NNI）

```python
def nearest_neighbor_index(self, area):
    """
    计算最近邻指数（Nearest Neighbor Index, NNI）
    :param area: 研究区域面积（平方公里）
    """
    # 使用KDTree查询最近邻
    tree = KDTree(self.points)
    distances, _ = tree.query(self.points, k=2)
    nearest_distances = distances[:, 1]  # 排除自身
    
    observed_mean_distance = np.mean(nearest_distances)
    density = self.n / area
    expected_mean_distance = 1 / (2 * np.sqrt(density))
    
    nni = observed_mean_distance / expected_mean_distance
    return nni, observed_mean_distance, expected_mean_distance
```

**NNI判据**：

| NNI值 | 分布模式 |
|-------|----------|
| NNI < 1 | 聚集分布 |
| NNI = 1 | 随机分布 |
| NNI > 1 | 均匀分布 |

### 2.2 网络维度（分形维数）

```python
def network_dimension(self):
    """计算网络维度（分形维数近似）"""
    all_coords = []
    for line in self.lines:
        coords = list(line.coords)
        all_coords.extend(coords)
    
    coords_array = np.array(all_coords)
    min_x, min_y = coords_array.min(axis=0)
    max_x, max_y = coords_array.max(axis=0)
    
    width = max_x - min_x
    height = max_y - min_y
    area = width * height
    total_len = sum(line.length for line in self.lines)
    
    if area > 0 and total_len > 0:
        scale = max(width, height)
        if scale > 0:
            dimension = np.log(total_len) / np.log(scale)
            return min(max(dimension, 1.0), 2.0)
    return 1.0
```

**分形维数含义**：
- D ≈ 1：线性结构，道路稀疏
- D ≈ 2：平面填充，道路密集

### 2.3 迂回指数

```python
def detour_index(self):
    """计算迂回指数（Detour Index）"""
    detour_sum = 0
    count = 0
    
    for line in self.lines:
        coords = list(line.coords)
        if len(coords) >= 2:
            start = Point(coords[0])
            end = Point(coords[-1])
            straight_dist = start.distance(end)
            if straight_dist > 0:
                detour_sum += line.length / straight_dist
                count += 1
    
    return detour_sum / count if count > 0 else 1.0
```

**数学原理**：

$$DI = \frac{1}{n} \sum_{i=1}^{n} \frac{L_i}{D_i}$$

- DI ≈ 1：道路较直
- DI > 1：道路曲折

class AreaDistributionAnalyzer:
    """面状分布测度分析器"""
    
    def __init__(self, polygon):
        self.polygon = polygon
        self.area = polygon.area
        self.perimeter = polygon.length

### 3.2 延伸率（Elongation Ratio）

```python
def elongation_ratio(self):
    """计算延伸率"""
    if self.area <= 0:
        return 1.0
    
    coords = self._get_all_coords()
    if len(coords) < 3:
        return 1.0
    
    centroid = np.mean(coords, axis=0)
    coords_centered = coords - centroid
    
    pca = PCA(n_components=2)
    pca.fit(coords_centered)
    
    eigenvalues = pca.explained_variance_
    if eigenvalues[0] > 0:
        ratio = np.sqrt(eigenvalues[1] / eigenvalues[0])
        return min(max(ratio, 0), 1)
    return 1.0
```

**原理**：通过PCA提取主成分，短轴与长轴之比

$$ER = \sqrt{\frac{\lambda_{min}}{\lambda_{max}}}$$

- ER ≈ 1：接近圆形
- ER ≈ 0：非常狭长

### 3.3 形态率（Form Ratio）

```python
def form_ratio(self):
    """计算形态率"""
    if self.area <= 0:
        return 0.0
    
    min_rect = self.polygon.minimum_rotated_rectangle
    rect_area = min_rect.area
    
    return self.area / rect_area if rect_area > 0 else 0.0
```

**原理**：面积与最小外接矩形面积之比

$$FR = \frac{A}{A_{rect}}$$

### 3.4 紧凑指数（Compactness Index）

```python
def compactness_index(self):
    """计算紧凑指数"""
    if self.area <= 0:
        return 0.0
    
    ideal_perimeter = 2 * np.sqrt(np.pi * self.area)
    return ideal_perimeter / self.perimeter if self.perimeter > 0 else 0.0
```

**原理**：等面积圆周长与实际周长之比

$$CI = \frac{2\sqrt{\pi A}}{P}$$

### 3.5 圆形率（Circularity Ratio）

```python
def circularity_ratio(self):
    """计算圆形率"""
    if self.perimeter <= 0:
        return 0.0
    
    return (4 * np.pi * self.area) / (self.perimeter ** 2)
```

**原理**：

$$CR = \frac{4\pi A}{P^2}$$

## 四、核心技术总结

### 4.1 依赖库说明

| 库 | 用途 |
|----|------|
| `numpy` | 数值计算、数组操作 |
| `scipy.spatial.KDTree` | 空间索引、最近邻查询 |
| `shapely` | 几何对象操作（Point/LineString/Polygon） |
| `sklearn.decomposition.PCA` | 主成分分析 |
| `matplotlib` | 数据可视化 |

### 4.2 关键算法复杂度

| 算法 | 时间复杂度 |
|------|------------|
| 平均中心 | O(n) |
| 中项中心 | O(n log n) |
| NNI（KDTree） | O(n log n) |
| PCA | O(n d²) |
| Boyce-Clark | O(n + k)，k为射线数 |

### 4.3 坐标系统注意事项

1. **统一坐标系**：所有数据应使用同一坐标系（如WGS84）
2. **投影转换**：面积和长度计算前需考虑投影
3. **单位一致性**：面积单位与长度单位需匹配
