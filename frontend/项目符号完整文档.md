## 一、TypeScript 类型符号

| 符号 | 名称 | 说明 | 项目示例 |
|------|------|------|----------|
| `interface` | 接口 | 定义对象的形状（结构） | `types/index.ts:1` |
| `type` | 类型别名 | 给类型起个别名 | 较少使用 |
| `:` | 类型注解 | 声明变量的类型 | `id: number` |
| `?` | 可选属性 | 表示属性可选 | `avatar?: string` |
| `\|` | 联合类型 | 类型 A 或类型 B | `role: 'admin' \| 'user'` |
| `[]` | 数组类型 | T 类型的数组 | `tags: Tag[]` |
| `<T>` | 泛型 | 可复用的类型模板 | `PaginatedResponse<T>` |
| `\| null` | 可空类型 | 允许为 null | `user: User \| null` |
| `typeof` | 类型查询 | 获取值的类型 | `typeof token === 'string'` |
| `extends` | 继承/扩展 | 接口继承或条件类型 | `interface A extends B` |
| `unknown` | 未知类型 | 安全的 any，需要类型判断 | `activities: unknown[]` |
| `any` | 任意类型 | 不检查类型（慎用） | `DefineComponent<{}, {}, any>` |
| `Partial<T>` | 部分类型 | T 所有属性变为可选 | `data: Partial<User>` |

---

### TypeScript 符号详解

#### 1. `interface` 接口
```typescript
// types/index.ts:1
export interface User {
  id: number
  username: string
  email: string
}
```

#### 2. `?` 可选属性
```typescript
// types/index.ts:6
export interface User {
  avatar?: string  // 可选
  nickname?: string
}
```

#### 3. `|` 联合类型
```typescript
// types/index.ts:5
role: 'admin' | 'user'  // 只能是 'admin' 或 'user'

// stores/user.ts:12
user: User | null  // User 类型或 null
```

#### 4. `[]` 数组类型
```typescript
// types/index.ts:19
tags: Tag[]  // Tag 类型的数组

// types/index.ts:51
replies?: Comment[]  // Comment 数组（可选）
```

#### 5. `<T>` 泛型
```typescript
// types/index.ts:82
export interface PaginatedResponse<T> {
  data: T[]  // T 类型的数组
  total: number
}

// 使用时
PaginatedResponse<Article>  // 文章分页
PaginatedResponse<User>     // 用户分页
```

#### 6. `Partial<T>` 部分类型
```typescript
// stores/user.ts:98
async updateProfileAction(data: Partial<User>) {
  // data 可以包含 User 的任意子集
}
```

## 二、JavaScript/TypeScript 运算符

### 2.1 逻辑运算符

| 符号 | 名称 | 说明 | 项目示例 |
|------|------|------|----------|
| `\|\|` | 逻辑或（短路） | 取第一个真值，否则最后一个 | `a \|\| b \|\| []` |
| `&&` | 逻辑与（短路） | 取第一个假值，否则最后一个 | `a && b` |
| `!` | 逻辑非 | 取反 | `!state.token` |
| `!!` | 双重非 | 转为布尔值 | `!!state.token` |

---

#### `||` 逻辑或
```typescript
// Home.vue:330
articles.value = response.articles || response.data || []
// 依次取 articles → data → 都没有就用 []

// request.ts:29
text: customConfig.loadingText || '加载中...'

// stores/user.ts:31
user: JSON.parse(localStorage.getItem('user') || 'null')
```

#### `&&` 逻辑与
```typescript
// Home.vue:335
if (article.coverImage && !article.coverImage.includes('fm=webp')) {
  // 只有两边都为 true 才执行
}

// Home.vue:187
<el-avatar :size="24" :src="userStore.user && userStore.user.avatar">
```

#### `!` 和 `!!` 逻辑非
```typescript
// stores/user.ts:59
isLoggedIn: (state) => !!state.token,
// !! 把 token 转为布尔值：null/undefined → false，其他 → true
// ! 单叹号 - 取反：true → false，false → true
// stores/user.ts:122
if (notification) {  // notification 存在
  notification.read = true
}
```

### 2.2 可选链和空值合并

| 符号 | 名称 | 说明 | 项目示例 |
|------|------|------|----------|
| `?.` | 可选链 | 安全访问嵌套属性 | `response.pagination?.total` |
| `??` | 空值合并 | 只对 null/undefined 生效 | `a ?? b`（项目中较少） |

---

#### `?.` 可选链
```typescript
// Home.vue:331
pagination.value.total = response.pagination?.total || response.total || 0
// 如果 response.pagination 是 null/undefined，整个表达式返回 undefined

// stores/user.ts:60

// 如果 state.user 是 null，返回 false
```

### 2.3 三元运算符

| 符号 | 名称 | 说明 | 项目示例 |
|------|------|------|----------|
| `? :` | 三元条件 | `条件 ? 值A : 值B` | `a > b ? a : b` |
                    条件 ? 值A : 值B
                    //   ↓        ↓    ↓
                    // true?   返回A : 返回B
                    // false?  跳过A : 返回B
---

#### `? :` 三元运算符
```typescript
// Home.vue:428
selectedCategory.value = selectedCategory.value === categoryId ? null : categoryId
// 如果相等则设为 null，否则设为 categoryId

// Home.vue:337
article.coverImage += article.coverImage.includes('?') ? '&fm=webp&q=80' : '?fm=webp&q=80'
```

### 2.5 比较运算符

| 符号 | 名称 | 说明 | 项目示例 |

| `>` `>=` | 大于/大于等于 | 比较大小 | `a > b`, `a >= b` |
| `<` `<=` | 小于/小于等于 | 比较大小 | `a < b`, `a <= b` |

## 三、Vue 模板符号

### 3.1 插值和指令

| 符号 | 名称 | 说明 | 项目示例 |
|------|------|------|----------|
| `{{ }}` | 插值表达式 | 显示数据 | `{{ article.title }}` |
| `v-if` | 条件渲染 | 条件为 true 时显示 | `v-if="loading"` |
| `v-else` | 否则 | 配合 v-if | `v-else` |
| `v-for` | 列表渲染 | 循环渲染 | `v-for="a in articles"` |
| `:key` | 循环 key | 唯一标识 | `:key="a.id"` |
| `:class` | 动态 class | 动态添加 class | `:class="{active: i==idx}"` |
| `@click` | 点击事件 | 监听点击 | `@click="fn"` |
| `:` | 属性绑定 | 动态属性 | `:src="a.src"` |

---

### Vue 模板符号详解

#### 1. `{{ }}` 插值表达式
```vue
<!-- Home.vue:78 -->
<span class="stat-number">{{ stats.articles }}</span>

<!-- Home.vue:138 -->
{{ cat.name }}
```

#### 2. `v-if` 和 `v-else` 条件渲染
```vue
<!-- Home.vue:34-44 -->
<template v-if="!userStore.isLoggedIn">
  <!-- 未登录显示 -->
</template>
<template v-else>
  <!-- 已登录显示 -->
</template>

<!-- Home.vue:148 -->
<ArticleSkeleton v-if="loading" />
<!-- loading 为 true 时显示骨架屏 -->
```

#### 3. `v-for` 列表渲染
```vue
<!-- Home.vue:104 -->
v-for="(banner, index) in bannerImages" 

<!-- Home.vue:152 -->
v-for="(article, index) in articles" 

<!-- Home.vue:132 -->
v-for="cat in categories" 

<!-- Home.vue:215 -->
v-for="tag in tags" 
```

#### 4. `:key` 循环 key
:key="唯一值" 作用 帮助 Vue 精确追踪每个元素 
唯一性 key 必须唯一（通常用 id） 
推荐 优先使用 item.id ，避免用 index （会增删时）
```vue
<!-- Home.vue:105 -->
:key="index"

<!-- Home.vue:154 -->
:key="article.id"
```

#### 5. `:class` 动态 class
作用 ：根据条件动态地给元素添加或移除 CSS class。
```vue
<!-- Home.vue:107 -->
:class="{ active: index === currentBannerIndex }"
<!-- index 等于 currentBannerIndex 时添加 active 类 -->

<!-- Home.vue:135 -->
:class="['category-tab', { active: selectedCategory === cat.id }]"
```

#### 6. `@click` 点击事件
```vue
<!-- Home.vue:124 -->
@click="currentBannerIndex = index"

<!-- Home.vue:136 -->
@click="handleCategoryClick(cat.id)"

<!-- Home.vue:219 -->
@click="handleTagClick(tag.name)"
```

#### 7. `:` 属性绑定
作用 ：根据条件动态地给元素添加或移除属性。
（如 src、alt、class 等）
（如 v-if、v-else 等）
```vue
<!-- Home.vue:109 -->
<img :src="banner.src" :alt="banner.alt" />

<!-- Home.vue:164 -->
<el-pagination
  :current-page="pagination.page"
  :page-size="pagination.limit"
  :total="pagination.total"
/>
```

## 四、Vue 3 Composition API 符号

| 符号 | 名称 | 说明 | 项目示例 |
|------|------|------|----------|
| `ref()` | 响应式引用 | 基本类型响应式 | `ref(0)` |
| `defineProps` | 组件 Props | 定义组件属性 | `defineProps<{a: T}>` |
| `defineEmits` | 组件事件 | 定义组件事件 | `defineEmits()` |

---

### Composition API 示例

#### 1. `ref()` 响应式引用
```typescript
// Home.vue:295
const articles = ref([])
const loading = ref(false)
const pagination = ref({ page: 1, limit: 10, total: 0 })

// 访问时需要加 .value
articles.value = response.data
loading.value = true
```

#### 2. `defineProps` Props 定义
```typescript
// ArticleCard.vue (示例)
const props = defineProps<{
  article: Article
  index: number
}>()
```

#### 3. `defineEmits` 事件定义
```typescript
// ArticleCard.vue (示例)
const emit = defineEmits<{
  like: [article: Article]
}>()

emit('like', article)  // 触发事件
```

## 八、常见符号组合

### 8.1 类型组合
```typescript
User | null              // 可空 User
Tag[] | []              // Tag 数组或空数组
Partial<User>           // User 的部分属性
'a' | 'b' | 'c'         // 字符串字面量联合
```

### 8.2 逻辑组合
```typescript
a?.b ?? c               // 可选链 + 空值合并
a && b || c             // 逻辑与 + 逻辑或
!!a                     // 转为布尔值
```

### 8.3 Vue 组合
```vue
v-for + :key            // 列表循环必备
:class="{ ... }"        // 动态 class
v-if + v-else           // 条件渲染
```
